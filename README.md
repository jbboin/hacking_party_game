# Birthday Hacker Game

## Quick Start
```bash
cd AI_birthday && npm start
```

## Find your IP
```bash
ipconfig getifaddr en0
```

## URLs
- Guest: `http://<YOUR_IP>:3000`
- Admin: `http://<YOUR_IP>:3000/admin`
- QR Code: `http://<YOUR_IP>:3000/qr`
- Scoreboard (TV): `http://<YOUR_IP>:3000/scoreboard`
